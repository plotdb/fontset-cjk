瀬戸フォント-SP v6.20

【入力方法など】
・スマホのシステムフォント等向け（？）に作った瀬戸フォント。
・Unicode文字は（全てではないです）入っていますが漢字を少なくして軽量化したフォントです。
・瀬戸フォントとの違いは、漢字を制限、ハングル（文字数が多いため）を入れていない点です。
